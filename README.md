## 2. 视频轨中的音频处理

### 2.1 MP4文件提取MP3

#### moviepy

提取MP4文件的MP3音频

- 安装包

```shell
pip install moviepy
```

- 单个mp4文件转化，调用`mp4_to_mp3.py`文件

```shell
python mp4_to_mp3.py -s './a.mp4' 
```

- 多个mp4文件转化，调用`mp4_to_mp3.py`文件

```shell
python mp4_to_mp3.py -s './mp4_dir' 
```
